# Sable AI Ltd — AI Governance Framework

**Live documentation site:** [davidcockson-compliance.github.io/sable-ai-governance-framework](https://davidcockson-compliance.github.io/sable-ai-governance-framework/)

An open-source AI governance and operational architecture framework for UK HR/recruitment AI providers. Built for **Sable AI Ltd**, a fictional early-stage UK startup whose product, **Scout**, is an AI-powered CV screening and candidate shortlisting tool.

**Coverage:** UK GDPR · Data (Use and Access) Act 2025 · Equality Act 2010 · DPA 2018 · ICO AI in Recruitment guidance (November 2024)

**18 documents across five stages** — AI system inventory, risk classification, data flow map, governance policy, GDPR compliance matrix, Equality Act 2010 discrimination map, ICO audit gap analysis, DPIA template, monitoring framework, bias monitoring protocol, incident response plan, data processing agreements, and candidate transparency notice. Plus four Phase 2 worked examples populated with Scout's confirmed technical architecture.
